# 🌦️ Previsão do Tempo

Projeto simples em HTML, CSS e JavaScript que consome a API do OpenWeather para exibir informações climáticas de uma cidade.

## 📌 Funcionalidades

* Buscar clima pelo nome da cidade
* Exibir:

  * Temperatura
  * Clima (descrição)
  * Umidade
  * Velocidade do vento

## 🛠️ Tecnologias utilizadas

* HTML
* CSS
* JavaScript
* API do OpenWeather

## ▶️ Como executar o projeto

1. Baixe ou clone o repositório
2. Abra a pasta no VS Code
3. Instale a extensão Live Server (se ainda não tiver)
4. Clique com o botão direito no arquivo `index.html`
5. Clique em **Open with Live Server**

## 🔑 Configuração da API

1. Acesse: https://openweathermap.org/api
2. Crie uma conta
3. Gere sua API Key
4. No arquivo `script.js`, substitua:

```javascript
const API_KEY = "SUA_CHAVE_AQUI";
```

## 📁 Estrutura do projeto

```
/projeto
│── index.html
│── script.js
```

## ⚠️ Observações

* O projeto utiliza unidades em Celsius
* O idioma da API está configurado para português
* Necessário conexão com internet para funcionar

## 📷 Exemplo de uso

Digite o nome de uma cidade e clique em "Buscar" para ver as informações do clima.
